# 游戏设计文档 (GDD)

## 1. 游戏概念

### 1.1 核心概念
- **类型**: 休闲 + 三消 + 模拟经营
- **平台**: 微信小游戏
- **目标用户**: 18-35岁，喜欢休闲游戏的玩家
- **游戏时长**: 单次5-15分钟，适合碎片时间

### 1.2 故事背景
玩家扮演一位年轻人，意外继承了远房亲戚留下的一座荒岛。岛上曾是繁华的度假胜地，但已废弃多年。玩家需要通过自己的努力，让这座小岛重焕生机。

## 2. 核心玩法

### 2.1 三消系统

#### 基础规则
- 8x8 的棋盘
- 6种基础元素（木材🪵、石头🪨、金币💰、星星⭐、爱心❤️、钻石💎）
- 滑动交换相邻元素
- 3个及以上相同元素连成一线即可消除

#### 消除奖励
| 消除数量 | 奖励 |
|---------|------|
| 3个 | 基础资源 |
| 4个 | 生成条形炸弹 |
| 5个(L/T形) | 生成炸弹 |
| 5个(直线) | 生成彩虹球 |

#### 道具效果
| 道具 | 效果 |
|------|------|
| 条形炸弹 | 消除整行或整列 |
| 炸弹 | 消除周围3x3区域 |
| 彩虹球 | 消除场上所有同类元素 |

#### 关卡目标类型
1. **收集目标**: 收集指定数量的某种元素
2. **分数目标**: 达到指定分数
3. **限时挑战**: 在时间内完成目标
4. **步数限制**: 在指定步数内完成

### 2.2 岛屿建设系统

#### 建筑类型
| 类型 | 建筑示例 | 功能 |
|------|---------|------|
| 住宅 | 小屋、别墅 | 增加人口上限 |
| 商业 | 商店、餐厅 | 产生金币 |
| 娱乐 | 游乐场、海滩 | 增加游客满意度 |
| 生产 | 伐木场、矿场 | 产生资源 |
| 装饰 | 花园、雕像 | 美化岛屿 |

#### 资源系统
| 资源 | 获取方式 | 用途 |
|------|---------|------|
| 木材 | 三消/伐木场 | 建造建筑 |
| 石头 | 三消/矿场 | 建造建筑 |
| 金币 | 三消/商业建筑 | 购买/升级 |
| 钻石 | 三消/任务奖励 | 高级道具 |

#### 升级系统
- 建筑可升级1-5级
- 升级需要资源和时间
- 高级建筑产出更多

### 2.3 剧情系统

#### NPC角色
1. **管家老王** - 引导教程，提供建议
2. **邻居小美** - 支线任务，好感度系统
3. **商人老陈** - 交易系统
4. **神秘旅客** - 主线剧情推进

#### 任务类型
- **主线任务**: 推进剧情，解锁新功能
- **支线任务**: NPC委托，获取奖励
- **日常任务**: 每日刷新，稳定收益

## 3. 游戏流程

### 3.1 新手引导
1. 剧情动画：继承小岛
2. 管家介绍基础操作
3. 第一关三消教学
4. 建造第一座建筑
5. 完成第一个任务

### 3.2 主循环
```
┌─────────────┐
│  进入游戏   │
└──────┬──────┘
       ↓
┌─────────────┐     ┌─────────────┐
│  岛屿主界面  │ ←── │  查看任务   │
└──────┬──────┘     └─────────────┘
       ↓
┌─────────────┐
│  进入三消   │
└──────┬──────┘
       ↓
┌─────────────┐
│  获取资源   │
└──────┬──────┘
       ↓
┌─────────────┐
│  建设岛屿   │
└──────┬──────┘
       ↓
┌─────────────┐
│  解锁内容   │
└──────┴──────┘
       ↓
     循环
```

## 4. 美术风格

### 4.1 整体风格
- 卡通Q版
- 明亮温暖的配色
- 治愈系画风

### 4.2 配色方案
- 主色: #4ECDC4 (海洋蓝绿)
- 辅色: #FFE66D (阳光黄)
- 强调: #FF6B6B (活力红)
- 背景: #F7FFF7 (清新白绿)

### 4.3 UI风格
- 圆角矩形按钮
- 柔和阴影
- 简洁图标

## 5. 音效设计

### 5.1 背景音乐
- 主界面: 轻快的海岛风格
- 三消关卡: 欢快有节奏感
- 建设界面: 悠闲放松

### 5.2 音效
- 消除音效: 清脆愉悦
- 建造音效: 锤子敲击
- 升级音效: 胜利号角
- 按钮点击: 轻柔反馈

## 6. 数值设计

### 6.1 体力系统
- 最大体力: 30
- 单关消耗: 5
- 恢复速度: 5分钟/1点
- 满体力时间: 2.5小时

### 6.2 关卡难度曲线
- 1-10关: 新手期，简单引导
- 11-30关: 成长期，逐渐增加难度
- 31-50关: 挑战期，需要策略
- 50+关: 进阶期，高难度挑战

### 6.3 经济平衡
| 阶段 | 金币产出/小时 | 主要消耗 |
|------|-------------|---------|
| 前期 | 100-300 | 基础建筑 |
| 中期 | 500-1000 | 升级建筑 |
| 后期 | 2000+ | 高级建筑 |
