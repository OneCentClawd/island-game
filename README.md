# 🏝️ 小岛物语 (Island Story)

一款休闲经营类微信小游戏，灵感来源于《逆袭的旅程》。

## 🎮 游戏概述

玩家继承了一座荒废的小岛，通过消除关卡获取资源，逐步建设并发展成繁华的度假胜地。

## 🎯 核心玩法

### 1. 三消玩法（资源获取）
- 连接3个及以上相同元素消除
- 获取建设资源（木材、石头、金币等）
- 道具系统（炸弹、彩虹球等）

### 2. 岛屿建设（经营模拟）
- 建造/升级各类建筑
- 解锁新区域
- 吸引游客，获取收益

### 3. 剧情系统
- NPC对话与任务
- 主线剧情推进
- 好感度系统

## 🛠️ 技术栈

- **引擎**: Cocos Creator 3.8 (免费)
- **语言**: TypeScript
- **后端**: 微信云开发 (免费额度)
- **美术**: 免费素材 + AI生成
